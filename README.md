# postpoint

一个 API 请求将消息发送到企业微信、飞书、钉钉的系统。

帮助你快速构建`消息推送`系统，发送`企业微信群机器人`、`钉钉自定义机器人`、`飞书自定义机器人`消息，可设置**失败重试**
机制，遵守各平台消息**频率限制**。

## 上手指南

请参考 [快速上手指南](https://github.com/lenye/postpoint/tree/main/docs/快速上手指南.md)

## 支持的操作系统

* Windows
* Linux
* macOS
* FreeBSD
